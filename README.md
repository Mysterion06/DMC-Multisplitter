# DMC-Multisplitter

This is a collection of all DMC Splitters put into one Splitter for a DMC Series run

## Features:
* Starts on DMC1
* Splits on every chapter for all DMC Games
* Final Splits for all games
* Timer stops when finishing a run, stops while a game is closed and resumes when the actual run is starting
* Fallback for extra splits that could happen
* Loadremover
